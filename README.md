# CinemaPlanner

CinemaPlanner is a web application to manage movies, halls, screenings, and bookings with poster storage.

## What you can do
- Browse and manage movies; upload posters that appear in lists and details.
- Manage halls and screenings; view upcoming screenings.
- See seating maps for screenings.
- Use the Operations dashboard for waiting lists, admin actions, and trailer playlists.
- Create bookings; see booking stats and admin contact.
- Check the home dashboard for quick access to movie counts, hall counts, screening counts, booking counts, and the next scheduled screening at a glance.
